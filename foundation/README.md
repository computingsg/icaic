# ICAIC Foundation (starter site)

Static starter site for `icaic.foundation`.

## What to edit first

- `index.html`: content, navigation, links
- `assets/styles.css`: colors and layout
- `assets/logo.svg`: logo mark

## Suggested next additions

- A dedicated page for `Rules`, `Timeline`, and `Participation`
- Sponsor / partner logos and a benefits page
- Governance / board page and legal footer
- Privacy policy and terms
